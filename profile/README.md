# Metadatics — Portable Batch Audio Metadata Editor

## About Metadatics

Metadatics provides a complete batch metadata editing environment for users who need to manage audio file tags. Metadatics combines batch editing, metadata import/export, online database lookup, and auto‑tagging from filename. Metadatics enables users to edit ID3 tags for MP3, FLAC, M4A, AAC, WAV, AIFF, OGG, and more. Metadatics runs on desktop platforms. Metadatics is trusted by music collectors, DJs, podcasters, audio engineers, and media library managers worldwide.

## Deep Dive — Metadata Editing Notes

Batch editing: edit multiple files at once, find/replace, add/remove, copy/paste
Auto‑tagging: extract metadata from filename, folder structure, music brainz
Online lookup: fetch tags from Discogs, MusicBrainz, iTunes, Amazon
Formats: MP3 (ID3v1, ID3v2), FLAC, M4A, AAC, WAV, AIFF, OGG, WMA
Artwork: fetch album art, add/remove/resize embedded images
